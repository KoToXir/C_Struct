# C_Struct
C语言练手项目
面向money编程，使用C语言手写其他高级语言的数据结果类型
初期版本会比较简单，



2021.12.07
基本完成map1.0版本，使用简单的结构体，几个指针链来链去

功能：
init，put，remove，get，（抽空再加个destory）

特点：
初始化大小用宏指定，可自由调整
结构体采用类似数组（连续的内存空间）内存储
容量到达0.75最大容量时自动扩容到1.5倍
put可覆盖
key和value均不可为NULL
删除数据时后边的数据会往前移（由于内存是连续的，在这个版本中如果数据不连续就乱套了）
